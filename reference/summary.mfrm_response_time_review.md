# Summarize a response-time review

Summarize a response-time review

## Usage

``` r
# S3 method for class 'mfrm_response_time_review'
summary(object, top_n = 10L, ...)
```

## Arguments

- object:

  An object returned by
  [`response_time_review()`](https://ryuya-dot-com.github.io/mfrmr/reference/response_time_review.md).

- top_n:

  Number of top groups to retain in summary previews.

- ...:

  Unused.

## Value

A list of class `summary.mfrm_response_time_review`.
